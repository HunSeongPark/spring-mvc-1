# spring-mvc-1  
스프링 MVC 1 - 상품 등록 예제를 통한 스프링 MVC 정리                 
강의 : [스프링 MVC 1편 - 백엔드 웹 개발 핵심 기술, 김영한 강사님](https://www.inflearn.com/course/%EC%8A%A4%ED%94%84%EB%A7%81-mvc-1)

* Dependencies
  - Spring Web
  - Thymeleaf
  - Lombok
